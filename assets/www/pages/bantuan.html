Bantuan

Apa yang harus saya lakukan jika saya mengalami masalah dengan aplikasi?

Jika Anda mengalami permasalahan dengan IMPRINT, silakan lakukan langkah-langkah berikut. Jika Anda membutuhkan bantuan silakan hubungi call-center kami.

1. Restart aplikasi IMPRINT.
2. Restart perangkat Anda.
3. Hapus cache aplikasi.
4. Hapus aplikasi dan data yang tidak diperlukan.
5. Install ulang aplikasi IMPRINT.

Keterangan :
Setelah melakukan beberapa langkah tersebut, kemungkinan Anda akan diminta untuk melakukan otorisasi ulang.

Apa yang harus saya lakukan jika saya lupa password ?

Jika Anda lupa dengan password Anda, silakan ikuti langkah-langkah berikut:

1. Klik tombol 'lupa password' di halaman login.
2. Jika Anda tidak memiliki informasi yang lengkap, maka Anda akan diminta untuk memasukkan alamat email. Selanjutnya, Anda akan menerima email notifikasi penggantian password.
3. Jika informasi Anda lengkap (termasuk nomor hp), maka password baru akan dikirimkan ke nomor hp Anda.

Mengapa saya tidak dapat menerima notifikasi ?

Jika Anda tidak dapat menerima notifikasi, pastikan beberapa kondisi Anda berikut :

- Koneksi internet Anda tidak terputus

Mengapa saya tidak dapat mendaftar dalam antrian baru ?

Jika Anda tidak dapat mendaftar dalam andivian baru, pastikan beberapa kondisi Anda berikut :

- Anda tidak sedang terdaftar dalam andivian manapun.
- Anda tidak dalam kondisi disuspend dalam andivian manapun
- Jika Anda memiliki kondisi tersebut di atas silakan batalkan andivian tersebut dahulu.

Apa yang harus saya lakukan jika saya tidak dapat datang di saat waktu pemeriksaan ?

Jika Anda tidak dapat hadir tepat waktu pada waktu pemeriksaan, andivian Anda aan disuspen oleh petugas maupun dokter sehingga andivian Anda akan terlewat dari daftar panggilan pasien.

Bagaimana cara untuk mengaktifkan kembali andivian saya setelah andivian saya di-suspend ?

Untuk mengaktifkan kembali andivian Anda, silakan menuju papan andivian dan klik tombol "aktifkan andivian"

Bisakah saya mengandivi di dua andivian yang berbeda di waktu yang sama?

Tidak. Demi menjaga ketertiban andivian dan agar tidak terjadi aksi saling serobot, Anda tidak diijinkan untuk mengandivi di dua antrian yang berbeda.

Bisakah saya melakukan booking waktu pemeriksaan untuk waktu yang akan datang?

Bisa. IMPRINT memungkinkan Anda untuk mem-booking waktu pemeriksaan untuk waktu yang akan datang. Sementara itu, Anda tidak dapat mendaftar dalam andivian baru lagi.
